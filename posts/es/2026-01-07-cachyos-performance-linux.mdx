## Contexto para Principiantes

> 💡 **Si ya conoces Linux y Arch**, puedes saltar directamente a la sección "¿Qué es CachyOS?".

### ¿Qué es una Distribución Linux?

Piensa en Linux como el "motor" de un coche. Una **distribución** (o "distro") es el coche completo: motor + carrocería + accesorios. Diferentes distros toman este mismo motor (Linux) y construyen coches diferentes — algunos enfocados en facilidad (Ubuntu), otros en ligereza (Linux Mint), y otros en rendimiento máximo (como CachyOS).

### ¿Qué es Arch Linux?

**Arch Linux** es una distribución conocida por ser "hazlo tú mismo". Te da control total, pero requiere conocimiento técnico para instalar y configurar. Es como construir una PC desde cero: tú eliges cada componente.

**CachyOS** toma esta poderosa base de Arch y añade:
- Un instalador fácil (no necesitas escribir comandos)
- Optimizaciones listas para tu hardware
- Configuraciones que "simplemente funcionan"

### ¿Qué es Rolling Release?

A diferencia de Windows que tiene versiones (Windows 10, 11, 12...), CachyOS usa el modelo **rolling release** — instalas una vez y recibes actualizaciones continuas para siempre. Nunca necesitas "formatear para actualizar".

## ¿Qué es CachyOS?

[CachyOS](https://cachyos.org/) es una distribución Linux que toma la base sólida de **Arch Linux** y añade optimizaciones agresivas de rendimiento. La diferencia está "bajo el capó".

El enfoque principal del proyecto es velocidad y estabilidad, logradas a través de optimizaciones en el **kernel** (el núcleo del sistema) y los paquetes de software.

## Principales Características Técnicas

¿Qué hace a CachyOS tan rápido? Aquí están los detalles:

### 1. Kernels Optimizados

El **kernel** es el núcleo del sistema operativo — hace de puente entre tus programas y el hardware (CPU, memoria, disco). CachyOS ofrece kernels personalizados con diferentes **schedulers** (programadores de tareas de CPU):

| Kernel | Scheduler | Ideal para |
|--------|-----------|------------|
| `linux-cachyos` | EEVDF | Uso general (predeterminado) |
| `linux-cachyos-bore` | BORE | Escritorio y Gaming |
| `linux-cachyos-bmq` | BMQ | Cargas de trabajo variadas |
| `linux-cachyos-rt-bore` | BORE + RT | Audio profesional |
| `linux-cachyos-lts` | EEVDF | Máxima estabilidad |

**En términos simples**: el scheduler decide qué programa usa la CPU primero. **BORE** es especialmente bueno para juegos porque prioriza la respuesta rápida del sistema. Puedes cambiar fácilmente entre kernels a través del **CachyOS Kernel Manager**.

### 2. Paquetes Optimizados para tu CPU

Esta es la característica más impresionante. CachyOS **recompila** los programas de Arch Linux con instrucciones específicas para procesadores modernos.

**Analogía**: Imagina que compras ropa de "talla única" — te queda, pero no es perfecta. CachyOS hace "ropa a medida" para tu CPU.

Detecta automáticamente tu arquitectura:

| Arquitectura | CPUs Compatibles | Beneficio |
|--------------|------------------|-----------|
| x86-64 | Todas (respaldo) | Compatibilidad |
| x86-64-v3 | Intel Haswell+ / AMD Ryzen 1+ | ~10-15% más rápido |
| x86-64-v4 | Intel Ice Lake+ / AMD Zen 4+ | ~15-20% más rápido |
| znver4 | AMD Zen 4 (Ryzen 7000) | Máxima optimización AMD |

**Términos técnicos explicados**:
- **LTO (Link Time Optimization)**: optimización al "ensamblar" el programa final
- **PGO (Profile-Guided Optimization)**: el programa se compila basándose en cómo la gente realmente lo usa
- **BOLT**: optimización adicional de Facebook/Meta para código aún más rápido

### 3. Sistema de Archivos Moderno

El instalador configura **BTRFS** por defecto — un sistema de archivos moderno con:
- **Compresión automática**: ahorra espacio en disco
- **Snapshots**: "fotos" del sistema que permiten volver atrás si algo sale mal
- **Mejor rendimiento** en SSDs y NVMe

### 4. Gaming de Primera Clase 🎮

CachyOS toma el gaming en serio:

- ✅ **Drivers NVIDIA** preconfigurados (propietarios y open-source)
- ✅ **Patches para Steam Deck** y **ROG Ally**
- ✅ **HDR habilitado** por defecto
- ✅ **Timer de 1000Hz** — menor latencia de entrada (presionas el botón, el juego responde más rápido)
- ✅ **Patches de Valve** para mejor compatibilidad con juegos

## Instalación: Más Fácil de lo que Parece

Olvida la fama de "Linux difícil". CachyOS ofrece un instalador gráfico intuitivo basado en **Calamares** (el mismo usado por distros "fáciles" como Manjaro).

### Opciones de Instalación

- **Online**: Descarga los paquetes más recientes (recomendado)
- **Offline**: Instala directamente desde la ISO (más rápido, pero los paquetes pueden estar desactualizados)

### Entornos de Escritorio Disponibles

CachyOS te deja elegir la "cara" de tu sistema:

| Escritorio | Estilo | Ideal para |
|------------|--------|------------|
| **KDE Plasma** | Moderno, personalizable (parece Windows 11) | La mayoría de usuarios |
| **GNOME** | Minimalista, amigable con touch | Quienes quieren simplicidad |
| **Hyprland** | Tiling, animaciones suaves | Usuarios avanzados |
| **XFCE** | Ligero, clásico | PCs más antiguas |
| **COSMIC** | Nuevo, de System76 | Quienes quieren experimentar |

Después de la instalación, **CachyOS Hello** te ayuda a configurar drivers, tweaks y aplicaciones.

## Primeros Pasos: Cómo Probarlo

¿Quieres experimentar? Sigue estos pasos:

### 1. Verifica tu Hardware

Descubre si tu CPU soporta x86-64-v3 o superior:

```bash
# En Linux, ejecuta este comando:
/lib/ld-linux-x86-64.so.2 --help | grep supported
```

### 2. Descarga la ISO

Visita [cachyos.org/download](https://cachyos.org/download/) y descarga la versión **KDE** (recomendada para principiantes).

### 3. Crea un USB Booteable

Usa [Ventoy](https://www.ventoy.net/) o [balenaEtcher](https://etcher.balena.io/) para grabar la ISO en el USB.

### 4. Prueba sin Instalar (Live USB)

Puedes ejecutar CachyOS directamente desde el USB para probarlo. Nada será modificado en tu PC hasta que hagas clic en "Instalar".

### 5. Instala (Opcional)

Si te gusta, usa el instalador gráfico. **Consejo**: instala en dual-boot con Windows si no quieres abandonar Windows todavía.

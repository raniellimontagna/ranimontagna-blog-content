# Posts en Español

Esta carpeta contiene las publicaciones del blog en español.

## Formato del archivo

Nomenclatura: `YYYY-MM-DD-slug-del-post.mdx`

Ejemplo: `2024-01-15-mi-primer-post.mdx`

⚠️ **Importante:** El slug (`mi-primer-post`) debe ser **exactamente el mismo** en todas las versiones de idioma (pt, en, es) para permitir el cambio de idioma en el blog.

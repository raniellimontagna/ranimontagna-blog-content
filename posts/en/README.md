# Posts in English

This folder contains blog posts in English.

## File format

Naming: `YYYY-MM-DD-post-slug.mdx`

Example: `2024-01-15-my-first-post.mdx`

⚠️ **Important:** The slug (`my-first-post`) must be **exactly the same** across all language versions (pt, en, es) to enable language switching in the blog.
